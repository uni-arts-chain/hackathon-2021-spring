# ERC1155

## Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`npx hardhat compile`

## Run Tests

`npx hardhat test`

## Deploy Contracts

`npm hardhat run scripts/deploy.ts`

## List Local Chain Accounts

`npx hardhat accounts`

## Run Hardhat Network

`npx hardhat node`

## Help

`npx hardhat help`
